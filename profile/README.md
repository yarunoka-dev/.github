# Yarunoka

Calendar-aware schedule DSL — schedules as data, not code.

Real-world schedules are calendar rules: "payday is the 25th, moved up
to the previous business day when it falls on a weekend or a holiday."
Yarunoka is a small JSON DSL (**Yrnk**) that states such rules as data,
plus engines that answer occurrence questions about them.

The official site is [yarunoka.dev](https://yarunoka.dev) —
documentation and the current list of language implementations. The
language-independent specification lives in
[spec](https://github.com/yarunoka-dev/spec); each language
implementation lives in its own repository in this organization.

The name is the Japanese question **やるのか？** (*yaru no ka?*) —
roughly "so, do we do it?". That is the question these engines exist to
answer.
